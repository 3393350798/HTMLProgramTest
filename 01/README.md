# 醇香咖啡 (Mellow Coffee) - 响应式网页前端

这是一个基于 HTML5 和 CSS3 开发的咖啡主题着陆页（Landing Page）。项目专注于优雅的视觉呈现和流畅的用户交互体验。

## 🚀 功能亮点

- **精致导航栏 (Header)**：
  - 品牌 Logo 与图标组合设计。
  - 响应式导航菜单，带有动态下划线悬停效果。
  - 功能按钮区：包含动态缩放的购物车图标、登录及购买按钮。
- **流畅交互体验**：
  - **无损缩放**：购物车图标使用 `transform: scale` 技术，缩放时不会抖动或干扰周围元素。
  - **胶囊按钮**：采用现代圆角设计（`border-radius: 50px`），配合平滑的颜色过渡效果。
  - **Hover 动效**：全局统一的 `transition` 过渡，提升用户操作的反馈感。
- **现代化 CSS 架构**：
  - 使用了最新的 **CSS 原生嵌套 (Nesting)** 语法，使样式结构更清晰、易于维护。
  - 采用 **Flexbox** 弹性盒模型，实现完美的垂直居中和间距控制。

## 🛠️ 技术栈

- **HTML5**：语义化页面结构。
- **CSS3**：
  - Flexbox 布局
  - CSS Nesting (嵌套)
  - CSS Variables (变量)
  - Transitions & Transformations
